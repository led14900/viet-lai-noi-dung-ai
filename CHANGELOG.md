# Changelog

Tất cả thay đổi quan trọng của plugin **Viết lại nội dung AI** được ghi lại trong file này.

## [1.0] - 2026-05-12

### Tính năng

- Cung cấp công cụ **Posts -> Viết lại nội dung AI** trong WordPress Admin.
- Hỗ trợ các provider:
  - OpenAI
  - Claude
  - Google Gemini
  - API tương thích OpenAI
- Thêm trang **Settings -> Viết lại nội dung AI** để cấu hình provider, model, temperature và thông tin thương hiệu.
- Thêm chức năng test kết nối từng provider.
- Lưu kết quả AI thành bản nháp WordPress để biên tập trước khi xuất bản.
- Lưu metadata gồm từ khóa chính, từ khóa phụ, số từ mục tiêu và ngày tạo.
- Hỗ trợ các cấu trúc nội dung: nội dung chuẩn SEO, Listicle, How-to Guide, Review và Comparison.
- Hỗ trợ các phong cách viết: chuyên nghiệp, thân thiện, hài hước, học thuật và thuyết phục.
- Mã hóa API key trước khi lưu vào database.
- Kiểm tra endpoint OpenAI-compatible để hạn chế URL không an toàn.
